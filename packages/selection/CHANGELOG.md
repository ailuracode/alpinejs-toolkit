# @ailuracode/alpine-selection

## 0.0.0

### Initial release

- Headless `SelectionController` with single, multiple, and range modes.
- Anchor and active item tracking.
- Toggle, replace, extend, clear, and select-all commands.
- Disabled item handling with optional override.
- Key-based selection independent of rendering order.
- Controlled and uncontrolled adapters.
- Optional serialization helpers for URL or storage integration.
