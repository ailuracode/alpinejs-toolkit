# @ailuracode/alpine-menu

## 0.2.0

### Minor Changes

- Add exclusive menu mode (default `true`): opening a menu closes all other open menus. Plugin option `exclusive` and per-menu `group` for menubar-style scoped exclusivity when `exclusive` is `false`. Scroll lock stays in sync when one menu replaces another.

## 0.1.0

### Minor Changes

- Add headless UI plugins: `x-child` directive plus dialog, menu, tooltip, tabs, accordion, command, and carousel stores.
