Consumers that depended on the triple-slash augmentation should declare it in their own `*.d.ts` or use `Alpine<{ theme: ThemeStore }>` from `@ailuracode/alpine-core`. Replace the package-local `ThemeAlpine` definition with the toolkit's generic `Alpine<{ theme: ThemeStore }>` (intersected with `cleanup?`). Fix `ThemeController` constructor calling `super({ id })` with an object instead of a string. Fix `local-storage.ts` type-only import. Build migrated from `tsc` + inline `node -e` to `tsup`. Remove unused `demo`/`demo:build`/`demo:preview` scripts.
