Breaking rewrite aligned to `@ailuracode/alpine-theme@0.2.x`. State property names renamed from `truly` / `falsely` / `ternary` to `on` / `off` / `indeterminate`. Method `cycle()` renamed to `next()`. `set(value)` now returns `void` instead of `boolean`. The controller now extends `BaseController` from `@ailuracode/alpine-core`, exposing `id`, `phase`, `isMounted`, `isDestroyed`, `mount()`, `destroy()`, and a typed `on('change', listener)` event bus with `{ current, previous, source }` detail payload. The default export `togglePlugin` is gone — use the named `togglePlugin(options?)` factory (matches `themePlugin`). The `createToggleMagic()` helper is gone — the plugin inlines the magic factory; standalone consumers use `createToggle(options)` which now returns a `ToggleController`. `@ailuracode/alpine-core` is now a peer dependency. Migration table in the [package README](https://github.com/ailuracode/alpinejs-toolkit/tree/main/packages/toggle#readme).
